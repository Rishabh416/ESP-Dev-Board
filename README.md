The ESP32 chip is the most versatile and easy to use chip in my opinion given its IoT functionality. 
I took on this project with the guide provided on the HackClub website to enhance my PCB skills as I have primarily only worked on simpler projects with less routing. 
This ESP dev board is unique since it has a male header allowing me to plug it directly into my PC rather than needing to stretch a cable for it.
